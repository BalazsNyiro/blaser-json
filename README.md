# blaser-json

Blaser JSON is a high-performance JSON parser written in Rust.

The project is built from scratch as a learning exercise and explores:

- finite-state machine based lexing
- JSON parsing
- Rust ownership and borrowing
- parser performance optimization
- benchmarking and testing
